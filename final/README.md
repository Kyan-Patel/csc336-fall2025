## About the App
RecipeBox is a simple web application that allows users to view and add recipes.

The app includes:
- A Home Page with navigation
- A Recipes Page that displays all saved recipes
- An Add Recipe Page with a form to submit new recipes

The backend stores recipe information in a JSON file and provides API routes for retrieving and adding recipes.
